# 1、什么是 PostCss

PostCss 是一个对 css 进行处理的插件，其最核心的功能就是将 css 解析成为一个 AST 语法树，再次基础上再对 AST 语法树进行进一步的封装，类似对 Css 针对不同的浏览器兼容，加上浏览器特定的浏览器标签头，就是对这个 AST 语法树，进行遍历。经过处理的树形结构最后再解析成为正常的 Css 语法渲染到页面中去。这就是常用的插件 autoprefixer, 在 PostCss 的基础上开发的，在此基础上已经开发了许多有用的插件。
