# 什么是 PostCss

PostCss 是一个对 css 进行处理的插件，其最核心的功能就是将 css 解析成为一个 AST 语法树，再次基础上再对 AST 语法树进行进一步的封装，类似对 Css 针对不同的浏览器兼容，加上浏览器特定的浏览器标签头，就是对这个 AST 语法树，进行遍历。经过处理的树形结构最后再解析成为正常的 Css 语法渲染到页面中去。这就是常用的插件 autoprefixer, 在 PostCss 的基础上开发的，在此基础上已经开发了许多有用的插件。

# RuckSack

  • 提供 css 函数，更方便的书写 css
  • 响应式的字体排版，仅需一个简单的 font-size: responsive 声明，就能够自动调整字体大小与行高。
  • 大量伪选择器，如 li:at-least (4)，选择了任意含至少 4 个条目的列表。
  • 属性别名，如在你的 CSS 中使用 bg 代替 background，

# 关于 postcss

PostCss 插件数量很多，学习成本很大，但是在开发中的帮助有很大的作用，在开发里力面有很多插件有重复的功能，所以要预防冲突。
现在还是在刚刚学习开发的阶段，要在实际开发中去逐步应用。
